# 수업 성찰 일지 (Class Reflection Journal)

## 프로젝트 개요

**수업 성찰 일지**는 한국 고등학교 영어 교사를 위한 수업 성찰 기록 및 분석 웹 애플리케이션입니다. 매 수업 후 정량적, 정성적 평가를 기록하고 누적된 데이터를 통해 수업 개선 방향을 도출할 수 있도록 설계되었습니다.

수능 영어 수업을 진행하는 교사가 자신의 수업을 체계적으로 돌아보고, 데이터 기반으로 교수법을 개선해 나가는 것을 핵심 목표로 합니다.

---

## 대상 사용자

| 구분 | 설명 |
|------|------|
| 주 사용자 | 한국 고등학교 영어 교사 |
| 수업 분야 | 수능 영어 대비 수업 |
| 기술 수준 | 일반 사용자 (별도 설치 불필요, 웹 브라우저만 있으면 사용 가능) |
| 사용 환경 | 데스크톱 및 모바일 브라우저 (반응형 디자인 지원) |
| 사용 시점 | 수업 직후 또는 수업 준비 시 이전 기록 참고 |

---

## 핵심 기능

### 1. 수업 성찰 기록 (Reflection Entry)

수업에 대한 종합적인 성찰 기록을 작성하는 핵심 기능입니다.

**기본 정보 입력:**
- 날짜 및 시간: 자동으로 현재 시간이 설정되며, 수동 수정 가능
- 수업 대상: 수업을 진행한 학급 정보 (예: 2학년 3반)
- 수업 주제: 해당 수업에서 다룬 핵심 주제 기록

**정량 평가 (5점 리커트 척도):**

5개 항목에 대해 1~5점 슬라이더로 평가합니다.

| 평가 항목 | 설명 |
|-----------|------|
| 수업 준비 충실도 | 수업 자료 및 계획 준비의 완성도 |
| 학생 집중도/참여도 | 수업 중 학생들의 참여 및 집중 수준 |
| 시간 관리 | 수업 시간 배분의 적절성 |
| 자기 만족도 | 교사 본인의 수업 만족 정도 |
| 에너지 레벨 | 수업 진행 시 교사의 에너지 및 활력 수준 |

**정성 평가 (서술형 기록):**

| 기록 항목 | 설명 |
|-----------|------|
| 수업 준비 과정 | 수업 전 준비한 내용과 과정에 대한 기록 |
| 잘한 점 | 수업에서 효과적이었던 부분에 대한 성찰 |
| 개선할 점 | 부족했던 부분과 보완이 필요한 영역 |
| 다음 수업 적용 사항 | 향후 수업에 반영할 구체적 행동 계획 |

### 2. 통계 및 분석 대시보드 (Statistics Dashboard)

누적된 성찰 데이터를 시각적으로 분석하여 수업 개선 방향을 제시합니다.

- **총 기록 수**: 전체 성찰 기록 횟수 표시
- **이번 달 기록**: 현재 월의 기록 횟수 추적
- **연속 기록일**: 매일 성찰을 작성한 연속 일수 (streak) 계산
- **전체 평균 점수**: 5개 평가 항목의 종합 평균
- **항목별 평균**: 각 평가 항목의 개별 평균 점수 및 시각적 바 그래프
- **추세 차트**: 최근 기록의 종합 평균 점수 변화를 막대 그래프로 시각화

### 3. 기록 이력 관리 (History Management)

작성된 모든 성찰 기록을 카드 형태로 열람하고 관리합니다.

- **카드 목록**: 날짜, 학급, 주제, 평균 점수가 한눈에 보이는 카드 레이아웃
- **상세 보기**: 각 카드를 펼쳐서 정량/정성 평가 전체 내용 확인
- **개별 삭제**: 확인 모달을 통한 안전한 기록 삭제
- **기록 수 표시**: 전체 기록 수를 헤더에 표시

### 4. 크로스 디바이스 동기화 (Cross-Device Sync)

Supabase 백엔드를 활용한 멀티 디바이스 데이터 동기화 기능입니다.

- **매직 링크 인증**: 비밀번호 없이 이메일만으로 로그인
- **양방향 동기화**: 로컬 데이터와 서버 데이터를 ID 기반으로 병합
- **동기화 상태 표시**: 실시간 동기화 상태를 시각적으로 표시 (연결됨/오프라인/동기화 중)
- **오프라인 지원**: 인터넷 연결 없이도 LocalStorage에 데이터 저장 후 재접속 시 자동 동기화

### 5. 데이터 관리 (Data Management)

기록 데이터의 백업 및 내보내기 기능을 제공합니다.

- **JSON 내보내기**: 전체 기록을 JSON 파일로 다운로드 (파일명에 날짜 자동 포함)
- **LocalStorage 영구 저장**: 브라우저 종료 후에도 데이터 보존
- **Supabase 클라우드 백업**: 로그인 시 서버에 자동 백업

## 사용 시나리오

### 시나리오 1: 수업 직후 성찰 기록

1. 수업을 마친 교사가 모바일 또는 PC에서 웹 앱을 엽니다
2. "성찰 기록" 탭에서 수업 대상과 주제를 입력합니다
3. 5개 항목의 슬라이더를 조절하여 정량 평가를 완료합니다
4. 서술형 필드에 수업 성찰 내용을 기록합니다
5. "저장" 버튼을 눌러 기록을 완료합니다

### 시나리오 2: 주간/월간 수업 분석

1. "통계" 탭을 열어 항목별 평균 점수를 확인합니다
2. 추세 차트를 통해 시간에 따른 수업 품질 변화를 파악합니다
3. 특정 항목의 점수가 낮은 경우 해당 영역의 개선 방안을 수립합니다

### 시나리오 3: 다른 기기에서 이어서 사용

1. 학교 PC에서 기록한 성찰 내용을 집 노트북에서 확인하고 싶습니다
2. 매직 링크 이메일로 로그인합니다
3. 자동 동기화를 통해 모든 기록이 동기화됩니다

### 시나리오 4: 데이터 백업

1. "이력" 탭에서 "내보내기" 버튼을 클릭합니다
2. 전체 기록이 JSON 파일로 다운로드됩니다
3. 필요 시 백업 파일을 안전한 곳에 보관합니다

## 가치 제안 (Value Proposition)

| 관점 | 가치 |
|------|------|
| 자기 개선 | 데이터 기반의 객관적 수업 개선 방향 도출 |
| 편의성 | 설치 없이 브라우저에서 바로 사용, 모바일 최적화 |
| 지속성 | 연속 기록 추적으로 성찰 습관 형성 동기 부여 |
| 접근성 | 크로스 디바이스 동기화로 언제 어디서나 기록 및 열람 |
| 데이터 소유권 | JSON 내보내기로 개인 데이터 완전한 제어 가능 |
| 비용 | 무료 사용 (Supabase 무료 티어 활용) |
